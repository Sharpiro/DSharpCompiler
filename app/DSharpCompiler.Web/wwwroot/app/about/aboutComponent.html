D# Compiler

Compiler Features

  • D# Static Subroutines
  • Call CLR static functions
  • Recursion
  • Conditional "eq" operator
  • Parameterized Functions
  • Scoped Variables
  • Return Values
  • Functions
  • Variables
  • Unary Operators: + -
  • Symbol Precedence
  • Binary Operators: + - * /
